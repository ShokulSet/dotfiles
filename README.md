# HOW TO USE
download chezmoi then run:
> chezmoi init --apply https://github.com/ShokulSet/dotfiles.git

## Forget how to use?
https://www.chezmoi.io/quick-start

## Bind
### Utility Bind
Screenshot to clipboard
> Mod+Tab

### Fn Bind
Mute audio
> Fn+F1

Lower volume
> Fn+F2

Raise volume
> Fn+F3

Lock screen
> Fn+F4

Lower brightness
> Fn+F5

Raise brightness
> Fn+F6
